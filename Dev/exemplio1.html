Principais tags

Como usar as tags de formas corretas

A Tag (br) serve para finalizar um paragrafo ou uma linha

A Tag (p) cria um paragrafo

A Tag (h1) e uma tag que so pode ser usada uma vez no codigo, pois ela indica algo importante no codigo

A tag (li) ele indica uma lista ordenada, como por exemplo a baixo, e sempre que for colocar uma ordem, tem que colocar o (li) em cada inicio de lista

A tag (ol) ela ordena a tag (li)

A tag (id) representa o identificador do elemento, voce pode colocar qualquer valor atribuido a essa ID

Exemplo da Tag (id) id="Qualquer coisa"

A tag (class) ela indica um texto

Exemplo tag (class), class="Texto qualquer"

A tag (a), representa um hiperlink, Exemplo, link que estão no mesmo site, para abrir uma pagina, ou uma URl de algum site expecifico, Mas a Tag (a) consequantemente tem a (href) junto para executar esse link
